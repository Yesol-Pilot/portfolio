🔗 **라이브 포털 접속**: [https://heoyesol.kr](https://heoyesol.kr)

## 🛠 기술 스택 (Tech Stack)

이 아키텍처는 고성능 모던 웹 스택을 기반으로 구축되었습니다.

### 코어 엔진 (Core Engine)

* ![React](https://img.shields.io/badge/-React_18-61DAFB?style=flat-square&logo=react&logoColor=black) **React 18** - 컴포넌트 아키텍처
* ![Vite](https://img.shields.io/badge/-Vite-646CFF?style=flat-square&logo=vite&logoColor=white) **Vite** - 초고속 빌드 도구
* ![Three.js](https://img.shields.io/badge/-Three.js-000000?style=flat-square&logo=three.js&logoColor=white) **R3F (@react-three/fiber)** - 3D 렌더링

### 비주얼 & 물리 (Visuals & Physics)

* **gsap** - 시네마틱 애니메이션
* **@react-three/postprocessing** - Bloom, Vignette, Glitch 효과
* **TailwindCSS** - UI 스타일링
* **Framer Motion** - 인터페이스 전환 효과

### 상태 관리 & 로직 (State & Logic)

* **Zustand** - 전역 상태 관리
* **Custom Hooks** - 오디오, 상호작용, 물리 엔진 훅

## 🚀 시작하기 (Quick Start)

로컬 환경에서 자신만의 차원을 생성해보세요.

### 필수 조건

* Node.js 18 이상
* npm

### 설치 방법

```bash
# 1. 저장소 복제 (Clone)
git clone https://github.com/ysh1537/portfolio.git

# 2. 디렉토리 이동
cd portfolio

# 3. 의존성 설치
npm install

# 4. 개발 서버 실행
npm run dev
```

`http://localhost:4321` 주소로 접속하여 멀티버스에 입장하세요.
